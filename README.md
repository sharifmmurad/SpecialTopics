## Portfolio 1 Reflection

For Portfolio 1, we were tasked with creating a student profile page for a school of our choice. The project consisted of two pages. The first page served as the home and welcome page, while the second page displayed the actual student profile information.

During this assignment, I learned how to structure multiple pages in a simple website and connect them together. One of the key features I implemented was creating my own button that allows the user to navigate back to the home page from the profile page.

This project helped me better understand page navigation, basic layout structure, and how user interface elements like buttons can improve the user experience when moving between pages.
